# ismailepereira.github.io

Site pessoal de **Ismaile Pereira Machado** — publicado automaticamente em
[ismailepereira.github.io](https://ismailepereira.github.io) via GitHub Pages.

Página estática (HTML + CSS, sem build, sem dependências) listando os projetos
que eu mantenho, todos open source no [@ismailepereira](https://github.com/ismailepereira).

## Adicionando um projeto novo

Edite `index.html`, copie um `<article class="project">` existente, troque
título, descrição, tags da stack e links. Commit e push — o GitHub Pages
atualiza em 1-2 min.
